# Iconos SVG

Este directorio debe contener los iconos SVG para los diferentes tipos de archivos.

## Iconos requeridos

- `generic.svg` - Icono genérico por defecto
- `pdf.svg` - Archivos PDF (opcional, ya hay preview real)
- `doc.svg` - Documentos Word (.doc, .docx, .rtf)
- `sheet.svg` - Hojas de cálculo (.xls, .xlsx, .csv)
- `slide.svg` - Presentaciones (.ppt, .pptx)
- `archive.svg` - Archivos comprimidos (.zip, .rar, .7z)
- `media.svg` - Archivos multimedia (.mp3, .wav, .flac, .mp4, .mkv, .mov)
- `code.svg` - Archivos de código (.py, .js, .ts, .html, .css)
- `exe.svg` - Ejecutables (.exe)

## Características recomendadas

- Tamaño base: 120x120px (se escalará automáticamente)
- Estilo: Moderno, minimalista, consistente
- Colores: Preferiblemente monocromático o con paleta limitada
- Formato: SVG válido
