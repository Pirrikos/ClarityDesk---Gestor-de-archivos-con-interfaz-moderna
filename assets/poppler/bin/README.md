# Binarios de Poppler

Este directorio debe contener los binarios de Poppler para Windows:

- `pdftoppm.exe` - Convierte PDF a imágenes (PPM/PNG)
- `pdftocairo.exe` - Convierte PDF usando Cairo
- `pdfinfo.exe` - Obtiene información de archivos PDF
- `pdftotext.exe` - Extrae texto de PDFs
- `pdfimages.exe` - Extrae imágenes de PDFs
- `pdftops.exe` - Convierte PDF a PostScript

## Descarga

Los binarios de Poppler para Windows se pueden descargar desde:
- https://github.com/oschwartz10612/poppler-windows/releases/

O desde la página oficial:
- https://poppler.freedesktop.org/

## Versión recomendada

Usar la última versión estable compatible con Windows 64-bit.
